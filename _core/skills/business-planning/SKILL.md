# 사업기획 스킬 (PDCA 기반)

## 용도
신규 사업 아이디어 발굴, 사업 기획, 실행, 점검, 개선의 전 과정을 PDCA 사이클로 관리할 때 사용합니다.
사업계획서, 제안서, 타당성 분석, 재무추정 등 사업 관련 모든 문서 작성을 지원합니다.

## PDCA 사업 실행 프레임워크

### Plan (기획)

#### 1. 기회 발굴
- 시장 트렌드 스캔 (PEST 분석)
- 고객 Pain Point 발굴 (인터뷰, 관찰)
- 경쟁 환경 분석 (Porter's 5 Forces)
- 자사 역량 분석 (SWOT, VRIO)
- 블루오션 전략 (전략캔버스, ERRC)

#### 2. 비즈니스 모델 설계
- 비즈니스 모델 캔버스(BMC) 9개 블록
  - 고객 세그먼트, 가치 제안, 채널, 고객관계
  - 수익원, 핵심자원, 핵심활동, 핵심파트너, 비용구조
- 린캔버스 (스타트업/신규 사업용)
  - 문제, 솔루션, 핵심지표, 경쟁우위, 고객, 채널, 비용, 수익

#### 3. 타당성 분석
- 시장 규모 추정 (TAM/SAM/SOM)
- 고객 검증 (인터뷰, 설문, PoC)
- 기술 타당성
- 재무 타당성 (손익분기점, ROI)
- 법률/규제 검토

#### 4. 사업계획서 작성
```
1. 사업 개요 (비전, 미션, 핵심 가치)
2. 시장 분석 (규모, 성장률, 트렌드)
3. 경쟁 분석 (경쟁사, 차별화 전략)
4. 제품/서비스 (핵심 가치 제안)
5. 마케팅/영업 전략 (GTM)
6. 운영 계획 (조직, 프로세스, 인프라)
7. 재무 계획 (3개년 추정, 손익, 현금흐름)
8. 리스크 관리 (리스크 매트릭스, 대응 계획)
9. 실행 로드맵 (마일스톤, WBS)
```

### Do (실행)

#### MVP 개발 및 검증
- 최소기능제품(MVP) 정의
- 빠른 프로토타이핑
- 파일럿 운영 (소규모 시장 테스트)
- 고객 피드백 수집

#### Go-to-Market 실행
- 채널 전략 실행
- 영업/마케팅 캠페인
- 파트너십/제휴 구축
- 초기 고객 확보

### Check (점검)

#### KPI 모니터링
| 영역 | 핵심 지표 |
|------|-----------|
| 재무 | 매출, 영업이익, 현금흐름, CAC, LTV |
| 고객 | 신규 고객 수, 이탈률, NPS, 재구매율 |
| 운영 | 납품 리드타임, 불량률, 가동률 |
| 학습 | 신규 역량 확보, 프로세스 개선 건수 |

#### 성과 리뷰
- 주간: 핵심 지표 대시보드 점검
- 월간: 실적 vs 계획 차이 분석
- 분기: 전략 방향성 점검, 피봇 여부 판단

### Act (개선)

#### 의사결정 매트릭스
- 성과 양호 + 시장 성장 → 스케일업 (투자 확대)
- 성과 부진 + 원인 명확 → 최적화 (프로세스 개선)
- 성과 부진 + 시장 변화 → 피봇 (전략 전환)
- 지속 적자 + 시장 축소 → 철수 (손절매)

#### 학습 문서화
- 성공/실패 요인 분석
- 프로세스 개선사항 기록
- 다음 PDCA 사이클에 반영

## 재무추정 가이드

### 손익추정 (P&L)
```
매출 = 단가 × 수량 × 채널 수
매출원가 = 직접비용 (인건비, 재료비, 외주비)
매출총이익 = 매출 - 매출원가
판관비 = 인건비 + 임차료 + 마케팅비 + 관리비
영업이익 = 매출총이익 - 판관비
```

### 3개년 추정 구조
| 항목 | Y1 (보수적) | Y2 (기본) | Y3 (낙관적) |
|------|------------|-----------|------------|
| 매출 | 시나리오별 | 시나리오별 | 시나리오별 |
| 비용 | 고정+변동 | 규모의 경제 | 최적화 |
| 손익 | 적자 허용 | 손익분기 | 흑자 목표 |
