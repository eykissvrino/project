# HR 종합 컨설팅 스킬

## 용도
HRM, HRD, OD, 인사제도 설계, 역량체계 구축 등 HR 컨설팅 전 영역의 업무를 수행할 때 사용합니다.
직무분석(job-analysis), 데이터분석(data-analysis), 보고서작성(consulting-report) 스킬과 조합하여 사용합니다.

## HR 컨설팅 프레임워크

### HRM (인적자원관리)

#### 인사제도 진단
1. 현행 제도 분석 (채용→배치→평가→보상→퇴직)
2. 제도 간 정합성 점검 (전략-제도-운영 Alignment)
3. 벤치마킹 (동종업계, 선진사례)
4. Gap 분석 및 개선 과제 도출

#### 직무체계 설계
- 직무분류체계 구축 (직군→직무→직무요소)
- 직무평가 (요소비교법, 점수법)
- 직무등급 및 직급체계 설계
- 직무기반 보상체계 연동

#### 평가체계 설계
- 성과평가: KPI/OKR 기반 목표관리
- 역량평가: 행동지표(BEI) 기반 다면평가
- 평가 프로세스: 목표설정→중간점검→연말평가→피드백
- 평가-보상-승진 연계 체계

#### 보상체계 설계
- 기본급: 직무급/성과급/연봉제
- 성과급: 개인/팀/조직 성과 연동
- 복리후생: 선택적 복리후생(Cafeteria Plan)
- 총보상 관점(Total Rewards) 설계

### HRD (인적자원개발)

#### 교육훈련 체계 설계
1. 교육 니즈 분석 (조직-직무-개인 수준)
2. 역량모델링 (핵심역량, 리더십역량, 직무역량)
3. 교육과정 체계 설계 (직급별×역량별 매트릭스)
4. 교육 운영 계획 (방법, 일정, 예산)
5. 교육 효과 측정 (Kirkpatrick 4단계)

#### 역량모델링 방법론
- 행동사건면접(BEI) 기반 역량 도출
- NCS 능력단위 매핑
- 역량사전(Competency Dictionary) 구축
- 역량수준 정의 (Level 1~5)
- 역량진단도구 개발

#### 경력개발(CDP) 설계
- 경력경로(Career Path) 설계
- 직무순환(Job Rotation) 체계
- 승진/전환 요건 정의
- 개인개발계획(IDP) 프레임워크

### OD (조직개발)
→ 상세 내용은 org-diagnosis 스킬 참조

## 컨설팅 프로세스 표준

```
1. 착수 (Kick-off)
   → 프로젝트 헌장, 이해관계자 분석, WBS 수립

2. 진단 (Diagnosis)
   → 현황 분석, 인터뷰, 설문, 벤치마킹

3. 설계 (Design)
   → To-Be 모델 설계, 대안 평가, 로드맵

4. 실행지원 (Implementation Support)
   → 파일럿, 교육, 변화관리

5. 정착 (Stabilization)
   → 모니터링, 피드백, 제도 보완
```

## 한국 HR 환경 필수 참조사항
- 근로기준법, 산업안전보건법 등 노동관계법
- NCS(국가직무능력표준) 체계
- 고용노동부 정책 (일터혁신, 산업전환 등)
- 한국직업능력연구원(KRIVET) 연구자료
- 산업별 인적자원개발위원회(ISC) 자료
- 사업주 직업능력개발훈련 지원 제도
