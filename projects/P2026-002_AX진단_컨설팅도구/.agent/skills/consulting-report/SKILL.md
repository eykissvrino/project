# 컨설팅 보고서 작성 스킬

## 용도
직무분석, 산업전환 진단, 훈련과정 개발 등 컨설팅 프로젝트의 보고서를 작성할 때 사용합니다.

## 보고서 기본 구조
1. **표지** — 프로젝트명, 수행기관, 발주처, 일자
2. **목차**
3. **사업 개요** — 배경, 목적, 범위, 추진체계
4. **수행 방법론** — 분석 프레임워크, 조사 방법, 일정
5. **분석 결과** — 핵심 발견사항, 데이터, 시사점
6. **제안/권고사항** — 구체적 실행 방안
7. **부록** — 설문지, 원시 데이터, 참고문헌

## 톤 & 스타일
- 공식적이고 전문적인 어조
- 약어 처음 사용 시 풀네임 병기 (예: NCS(국가직무능력표준))
- 핵심 내용은 표나 도식으로 시각화
- 페이지 번호, 머리글/바닥글 포함

## 파일명 규칙
`[유형]_[기관명]_[프로젝트명]_[버전].확장자`
예: `결과보고서_한화연_산업전환진단_v2.1.pptx`
