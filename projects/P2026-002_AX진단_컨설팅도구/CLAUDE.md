# P2026-002 AX(인공지능전환) 진단 컨설팅 도구 개발

## 개요
- **클라이언트**: 컨소시엄 공동훈련 참여기업
- **기간**: 2026년
- **목적**: 기업의 AI 전환(AX) 수준을 진단하고, 맞춤형 컨설팅 도구를 개발하여 참여기업의 AI 전환 역량을 강화

## 주요 산출물
- [ ] AX 진단 설문지/인터뷰 가이드
- [ ] AX 진단 결과 분석 도구 (Excel/웹)
- [ ] AX 진단 컨설팅 보고서 템플릿
- [ ] 기업별 맞춤 AX 전환 로드맵

## 핵심 에이전트
- **Hera**: 조직 진단, 역량 분석
- **Apollo**: 벤치마킹, 데이터 분석
- **Hermes**: 보고서/산출물 작성

## 작업 규칙
- 모든 문서는 한국어로 작성
- 최종 산출물은 `outputs/` 폴더에 저장
- 참고 자료는 `data/` 폴더에 저장
- 작업 기록/메모는 `docs/` 폴더에 저장

## 컨텍스트
- AX = 인공지능전환 (AI Transformation)
- 컨소시엄 공동훈련 사업 내 참여기업 대상 진단 도구
- NCS 기반 역량 체계와 연계 가능
- 참고자료(PDF)는 `참고자료/` 폴더에 보관
