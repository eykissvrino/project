# 경영전략 수립 스킬

## 용도
중장기 경영전략 수립, 사업 포트폴리오 분석, 성과관리 체계 설계 등 경영기획/전략 관련 업무에 사용합니다.

## 전략 수립 프레임워크

### 전략 수립 프로세스

```
1. 환경 분석 (Where are we?)
   → 외부환경: PEST, 5 Forces, 산업 동향
   → 내부환경: SWOT, 가치사슬, 핵심역량
   → 이해관계자 분석

2. 전략 방향 설정 (Where do we want to be?)
   → 비전/미션 재정립
   → 전략 목표 설정 (3~5년)
   → 전략 방향 도출

3. 전략 과제 도출 (How do we get there?)
   → 전략 맵 작성 (BSC 4관점)
   → 전략 과제 도출 및 우선순위화
   → 실행 로드맵

4. 실행 체계 설계 (How do we execute?)
   → KPI/OKR 체계
   → 자원 배분 계획
   → 모니터링 체계

5. 성과 관리 (How do we track?)
   → 정기 성과 리뷰
   → 전략 수정/보완
   → 학습 및 환류
```

### 분석 도구

#### 외부 환경 분석

**PEST 분석**
| 영역 | 분석 항목 |
|------|-----------|
| Political (정치) | 정부 정책, 규제, 노동법 변화, 보조금 |
| Economic (경제) | GDP, 금리, 환율, 산업 성장률 |
| Social (사회) | 인구 구조, 소비 트렌드, 가치관 변화 |
| Technological (기술) | AI, 자동화, 디지털 전환, 신기술 |

**Porter's 5 Forces**
- 기존 경쟁자 간 경쟁 강도
- 신규 진입자의 위협
- 대체재의 위협
- 공급자의 교섭력
- 구매자의 교섭력

#### 내부 환경 분석

**SWOT 분석**
- Strengths (강점): 자사의 차별화 역량
- Weaknesses (약점): 개선이 필요한 영역
- Opportunities (기회): 활용 가능한 외부 기회
- Threats (위협): 대비해야 할 외부 위협
- TOWS 매트릭스로 전략 대안 도출 (SO/ST/WO/WT)

**가치사슬 분석 (Value Chain)**
- 본원적 활동: 물류→운영→물류→마케팅→서비스
- 지원 활동: 인프라, HRM, 기술개발, 조달

### 성과관리 체계

#### BSC (Balanced Scorecard)
| 관점 | 핵심 질문 | KPI 예시 |
|------|-----------|----------|
| 재무 | 주주에게 어떤 가치를? | 매출, 영업이익률, ROE |
| 고객 | 고객에게 어떤 가치를? | 고객만족도, NPS, 점유율 |
| 내부 프로세스 | 어떤 프로세스를 탁월하게? | 프로세스 효율, 품질, 혁신 |
| 학습/성장 | 어떻게 역량을 키울? | 역량 수준, 교육시간, 이직률 |

#### OKR (Objectives & Key Results)
```
Objective: 달성하고자 하는 목표 (정성적, 영감을 주는)
  Key Result 1: 측정 가능한 결과 1 (정량적)
  Key Result 2: 측정 가능한 결과 2
  Key Result 3: 측정 가능한 결과 3

원칙:
- Stretch Goals (도전적 목표)
- 70% 달성이 성공
- 분기별 갱신
- 상향식 + 하향식 결합
```

### 사업 포트폴리오 분석

#### BCG 매트릭스
| 구분 | 시장성장률 | 상대적 점유율 | 전략 |
|------|-----------|--------------|------|
| Star (스타) | 높음 | 높음 | 집중 투자 |
| Cash Cow (현금젖소) | 낮음 | 높음 | 현금 회수 |
| Question Mark (물음표) | 높음 | 낮음 | 선택적 투자 |
| Dog (개) | 낮음 | 낮음 | 철수/매각 |

#### GE-McKinsey 매트릭스
- 산업 매력도 (시장 규모, 성장률, 수익성, 경쟁 강도)
- 경쟁적 강점 (점유율, 브랜드, 기술력, 비용 우위)
- 9개 셀 포트폴리오 전략

## 전략 문서 표준 구조

### 경영전략 보고서
```
1. Executive Summary
2. 경영 환경 분석
   - 외부 환경 (PEST, 산업 동향)
   - 내부 환경 (SWOT, 역량 진단)
3. 전략 방향
   - 비전/미션
   - 전략 목표
   - 전략 방향 (3~5개)
4. 전략 과제
   - 과제별 상세 내용
   - 우선순위 매트릭스 (시급성×영향도)
5. 실행 계획
   - 과제별 로드맵
   - 자원 배분
   - 성과 지표 (KPI/OKR)
6. 모니터링 체계
   - 성과 리뷰 주기/방법
   - 전략 수정 프로세스
```
