# 데이터 분석 스킬

## 용도
설문 분석, 직무분석 데이터 처리, HR 애널리틱스 등 데이터 분석 작업에 사용합니다.

## 분석 유형별 접근

### 설문조사 분석
1. 데이터 정제 (결측치, 이상치 처리)
2. 기술통계 (빈도, 평균, 표준편차)
3. 교차분석 (부서별, 직급별, 연차별)
4. 신뢰도 분석 (Cronbach's α)
5. 시각화 (차트, 그래프)

### 직무분석 데이터
1. Task별 빈도/중요도/난이도 정리
2. 직무요소 우선순위 도출 (빈도×중요도)
3. 역량 Gap 분석
4. 교육 니즈 도출

### HR 데이터
1. 인력 현황 분석 (인구통계, 이직률)
2. 역량 수준 분석
3. 교육훈련 효과 분석
4. 벤치마킹 비교 분석

## 산출물
- 분석 결과: xlsx (원시 데이터 + 분석 시트 + 요약)
- 시각화: pptx (차트 중심 보고자료)
- 분석 코드: .py (재현 가능한 분석 스크립트)

## 도구
- Python: pandas, matplotlib, seaborn
- Excel: 피벗테이블, 조건부서식, 차트
