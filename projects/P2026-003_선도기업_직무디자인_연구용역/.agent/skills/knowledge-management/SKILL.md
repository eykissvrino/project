# 지식관리·학습 스킬

## 용도
MBA 학습, AI 활용 역량 개발, HR/경영 지식 축적 등 지속적인 학습과 지식 관리에 사용합니다.
학습 노트 작성, 지식 체계화, 인사이트 정리, 학습 계획 수립을 지원합니다.

## 학습 노트 작성 프레임워크

### Zettelkasten (제텔카스텐) 방식
1. **원자적 노트**: 하나의 노트에 하나의 아이디어
2. **연결**: 관련 노트 간 링크로 지식 네트워크 구축
3. **자기 언어**: 원문 요약이 아닌 자신의 말로 재구성
4. **출처 기록**: 원본 서적/논문/강의 출처 명시

### 학습 노트 구조
```markdown
# [주제]

## 핵심 개념
- 개념 정의 (한 문장)
- 핵심 키워드 3~5개

## 내용 정리
- 핵심 내용을 자기 언어로 정리
- 사례/예시 포함

## 실무 적용
- 현재 프로젝트/업무에 어떻게 적용할 수 있는가
- 구체적 액션 아이템

## 연결 고리
- 관련 개념: [링크]
- 관련 프로젝트: [링크]
- 추가 학습 필요: [주제]

## 출처
- 저자, 제목, 연도, 페이지
```

#### 학습 전략
1. **이론→실무 연결**: 모든 이론을 실제 프로젝트에 적용 시도
2. **케이스 스터디**: 하버드/SKK GSB 케이스로 분석력 강화
3. **논문 리딩**: 학술 논문 월 2편 이상 정리
4. **토론/발표**: 스터디 그룹 활용

## 지식 관리 체계

### 폴더 구조 (프로젝트 내)
```
프로젝트/docs/
├── learning/          ← 학습 노트
│   ├── mba/          ← MBA 과목별 정리
│   ├── ai/           ← AI 학습 정리
│   ├── hr-trends/    ← HR 트렌드 정리
│   └── books/        ← 독서 노트
├── insights/          ← 인사이트 메모
└── references/        ← 참고자료 링크/요약
```

### 지식 축적 사이클
```
수집 → 정리 → 연결 → 적용 → 회고
 ↑                              │
 └──────────────────────────────┘
```

1. **수집**: 강의, 논문, 도서, 뉴스에서 핵심 내용 포착
2. **정리**: 학습 노트 형식으로 자기 언어로 재구성
3. **연결**: 기존 지식/프로젝트와 연결 고리 형성
4. **적용**: 실제 업무/프로젝트에 적용
5. **회고**: 적용 결과 리뷰, 학습 내용 보완
