# 직무분석 스킬

## 용도
NCS 기반 직무분석, 직무기술서 작성, 역량체계 도출 등 직무분석 관련 작업에 사용합니다.

## 직무분석 프레임워크

### 1단계: 직무 분류체계 구축
- 직무군 → 직무 → 직무요소(Task) → 작업활동(Activity)
- NCS 분류체계 참조: 대분류 → 중분류 → 소분류 → 세분류

### 2단계: 직무정보 수집
- 워크숍/인터뷰 (현직자, 관리자, SME)
- 설문조사 (빈도, 중요도, 난이도)
- 기존 문서 분석 (직무기술서, 업무매뉴얼)

### 3단계: 직무기술서 작성
각 직무요소(Task)별:
- 작업명
- 작업 설명
- 수행 빈도 / 중요도 / 난이도
- 필요 지식(Knowledge)
- 필요 기술(Skill)
- 필요 태도(Attitude)
- 관련 NCS 능력단위

### 4단계: 역량 도출
- 직무별 핵심역량 매핑
- 역량 수준 정의 (초급/중급/고급)
- 역량 개발 경로 설계

## 산출물 형식
- 직무분류체계도: pptx (시각적 다이어그램)
- 직무분석 워크시트: xlsx (Task별 상세 정보)
- 직무기술서: docx 또는 hwp
- 역량체계도: pptx
