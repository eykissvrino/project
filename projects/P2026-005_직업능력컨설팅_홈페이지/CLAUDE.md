# P2026-005 직업능력컨설팅본부 홈페이지 구축

## 개요
- **클라이언트**: 시앤피컨설팅 직업능력컨설팅본부
- **기간**: 2026년
- **목적**: 시앤피컨설팅 직업능력컨설팅본부의 서비스 소개 및 포트폴리오를 보여주는 홈페이지 구축

## 기술 스택
- **프레임워크**: Next.js (App Router)
- **스타일**: SCSS Modules
- **배포**: 정적 사이트 (outputs/ 폴더에 빌드 결과물)

## 주요 산출물
- [x] 홈페이지 소스코드 (outputs/source/)
- [x] 정적 빌드 결과물 (outputs/)
- [ ] 도메인 연결 및 배포

## 핵심 에이전트
- **Daedalus**: 개발 총괄 (Arachne=FE)
- **Aphrodite**: 브랜딩, 콘텐츠 기획

## 페이지 구성
- 메인 (/) — 회사 소개, 핵심 서비스
- HRM (/hrm) — 인적자원관리 서비스
- HRD (/hrd) — 인적자원개발 서비스
- AX 컨설팅 (/ax-consulting) — AI 전환 컨설팅
- 포트폴리오 (/portfolio) — 수행 실적
- 연락처 (/contact) — 문의

## 작업 규칙
- 소스코드는 `outputs/source/`에 위치
- 빌드 결과물은 `outputs/`에 위치
- 모든 텍스트는 한국어

## 컨텍스트
- 시앤피컨설팅 = HR 컨설팅 전문 기업
- 주요 서비스: HRM(인적자원관리), HRD(인적자원개발), AX(인공지능전환) 컨설팅
