# 🚀 장애인 B2B 고용 솔루션 프로젝트 대시보드

## 📖 프로젝트 개요
본 프로젝트는 기업의 장애인 고용 의무 부담금 문제를 해결하고, 실질적이고 지속 가능한 직무 가치를 창출하기 위한 **종합 HR BPO(Business Process Outsourcing) 솔루션**입니다. 
단순한 인력 파견을 넘어 **게이미피케이션 기반 온보딩 워크샵**, **전담 관리자 서비스**, **맞춤형 보조공학 지원** 등을 체계적인 패키징(Basic, Standard, Enterprise)으로 제공하여 압도적인 고객 경험을 제공합니다.
