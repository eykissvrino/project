# 🏁 M1~M6 마일스톤 기반 AI 부서별 액션 플랜 (Q1-Q2)

본 문서는 `04_2026_Pilot_Milestone_Plan.md`에서 정의된 상반기(1월~6월) 마일스톤을 달성하기 위해, **어떤 AI 에이전트 부서가 언제까지 무엇을(명확한 산출물) 만들어 내야 하는지** 구체적으로 할당한 실행 지시서입니다. 

---

## 📅 [Step 0] 비즈니스 브랜딩 파운데이션 (극초단위 기획)
*모든 업무의 베이스캠프가 될 기업의 존재 이유와 얼굴을 세팅하는 "Day 1~5" 핵심 기초 업무입니다.*

* **담당 부서:** `01_Strategy_and_Planning` & `03_Accessibility_and_Design`
* **즉각 실행 마이크로 태스크 (Action Items):**
  - [ ] **Task 1. 네이밍(Naming) 및 도메인:** 직관적 혹은 은유적인 기업명 후보 10개 도출, 도메인(.com/co.kr) 확보 가능 여부 확인 후 1개 최종 낙점.
  - [ ] **Task 2. 미션/비전 선언문 (Mission & Vision):** "왜 우리가 존재하는가"와 "우리는 어떤 세상을 만들 것인가"에 대한 1페이지 공식 결의문 작성.
  - [ ] **Task 3. 브랜드 핵심 가치 (Core Values):** 조직 문화와 서비스 퀄리티를 대변하는 3대 핵심 가치(e.g., 혁신성, 책임감 등) 캐치프레이즈 정의.
  - [ ] **Task 4. 30초 엘리베이터 피치 (Elevator Pitch):** 누군가 우리 회사가 무얼 하는 곳인지 물었을 때, 단숨에 꽂히는 30초 구두 설명 대본 작성.
  - [ ] **Task 5. 브랜드 아이덴티티 및 비주얼 가이드:** 로고 컨셉안, 메인/서브 브랜드 컬러 코드(Hex), 브랜드 톤앤매너 1장 압축 가이드 생성.

* **담당 부서:** `02_Sales_and_Marketing` & `03_Accessibility_and_Design`
* **즉각 실행 마이크로 태스크 (Action Items):**

  - [x] **[영업팀] 마케팅 키 메시지 / 슬로건 정의:** B2B 고객의 이목을 끌 수 있는 1줄짜리 후킹 카피 5종 A/B 테스트 세트 기획.
  - [x] **[디자인팀] 보드게임 실물 시안용 이미지 프롬프트 작성:** Type A, C 보드게임 기획서를 바탕으로 일러스트/에셋 제작에 필요한 AI 이미지 프롬프트 상세 작성 (게임판, 카드 등).

### 🚩 M2 (2월): 법인 설립 및 규제/인허가 클리어
* **담당 부서:** `01_Strategy_and_Planning`
* **즉각 실행 마이크로 태스크 (Action Items):**

  - [ ] **[기획팀] 법인 설립을 위한 초기 자본금/예산안 정리표 작성:** 최소한의 B2B 런칭에 필요한 초기 예산 분석.

### 🚩 M3 (3월): B2B 마케팅용 인바운드 채널 구축
* **담당 부서:** `05_IT_Engineering`
* **즉각 실행 마이크로 태스크 (Action Items):**
  - [ ] **[IT팀] 과징금 ROI 시뮬레이터 로직 작성:** "종업원 수 X명 적용 시 예상 과징금 Y원 산출" 프로그램 로직(JS 함수 등) 코드 블록 초안 작성.
  - [ ] **[IT팀] 인바운드 홍보용 랜딩페이지 UI 코드 / 기획안 작성:** 고객이 시뮬레이터를 이용하고 연락처(DB)를 남기게 유도하는 1페이지짜리 프론트엔드 랜딩페이지(HTML/React 구조 등) 기획안 작성.
