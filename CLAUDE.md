# 시앤피컨설팅 AI 워크스페이스

## 개요
이석주의 통합 AI 프로젝트 워크스페이스입니다.
Claude Code, Antigravity 등 AI 도구를 사용할 때 이 폴더 안에서 작업합니다.

## 구조
- `_shared/` — 모든 프로젝트에서 공유하는 스킬, 템플릿, 프롬프트
- `projects/` — 활성 프로젝트 (프로젝트별 독립 폴더)
- `archive/` — 완료된 프로젝트
- `outputs/` — 최종 산출물 모음 (클라이언트 전달용)

## 작업 규칙
- 모든 응답과 문서는 **한국어**로 작성
- 새 프로젝트 시작 시: `_shared/templates/project-template/`를 복사하여 사용
- 최종 산출물은 프로젝트 내 `outputs/`에 저장하고, 필요 시 루트 `outputs/`에도 복사
- 컨설팅 산출물은 전문적이고 공식적인 톤 유지

## 자주 쓰는 산출물 유형
- 직무분석 보고서 (pptx, xlsx)
- 제안서/발표자료 (pptx)
- 훈련과정 로드맵 (pptx, xlsx)
- 컨설팅 결과보고서 (pptx, pdf)
- 데이터 분석 결과 (xlsx)
