# 조직진단·조직설계 스킬

## 용도
조직진단, 조직설계, 조직문화 분석, 변화관리 등 OD(조직개발) 관련 컨설팅 업무에 사용합니다.
HR 컨설팅(hr-consulting) 스킬의 OD 영역을 상세화한 전문 스킬입니다.

## 조직진단 프레임워크

### 진단 모델

#### McKinsey 7S 모델
| 요소 | 진단 항목 | 분석 방법 |
|------|-----------|-----------|
| Strategy (전략) | 비전, 미션, 전략 방향 | 경영계획 분석, 경영진 인터뷰 |
| Structure (구조) | 조직도, 의사결정 체계, 권한 위임 | 조직도 분석, 직책/직급 체계 |
| Systems (시스템) | 인사제도, 정보시스템, 업무 프로세스 | 제도 문서 분석, 프로세스 맵핑 |
| Shared Values (공유가치) | 핵심가치, 조직문화 | 문화 진단 설문, FGI |
| Style (리더십) | 리더십 스타일, 의사소통 방식 | 리더십 진단, 360도 피드백 |
| Staff (인력) | 인력 구성, 역량 수준 | 인력 현황 분석, 역량 진단 |
| Skills (역량) | 핵심역량, 기술 수준 | 역량체계 분석, 벤치마킹 |

#### Burke-Litwin 변화 모델
- 전환적 요소: 외부환경, 미션/전략, 리더십, 조직문화
- 거래적 요소: 구조, 시스템, 관리관행, 분위기, 직무/역할
- 결과: 개인/조직 성과

#### Weisbord 6-Box 모델
1. 목적 (Purpose)
2. 구조 (Structure)
3. 관계 (Relationships)
4. 보상 (Rewards)
5. 리더십 (Leadership)
6. 지원 메커니즘 (Helpful Mechanisms)

### 진단 수행 절차

```
1. 진단 설계
   → 진단 모델 선정, 진단 범위/대상 확정
   → 진단 도구 설계 (설문, 인터뷰 가이드)

2. 데이터 수집
   → 정량: 설문조사, HR 데이터, 재무 데이터
   → 정성: 인터뷰(1:1), FGI, 워크숍, 관찰

3. 분석
   → 현황 분석 (As-Is)
   → 문제 구조화 (Issue Tree)
   → 근본 원인 분석 (Root Cause)
   → 벤치마킹 비교

4. 결과 도출
   → 핵심 이슈 정리
   → 시사점 도출
   → 개선 방향 제시
```

## 조직설계 프레임워크

### 조직구조 설계
- 기능별 조직 (Functional)
- 사업부제 (Divisional)
- 매트릭스 조직 (Matrix)
- 프로젝트 조직 (Project-based)
- 네트워크 조직 (Network)
- 애자일 조직 (Agile/Squad)

### 설계 원칙
1. **전략-구조 정합성**: 전략이 구조를 결정 (Chandler)
2. **적정 통솔범위**: 관리자당 직접 보고 인원 최적화
3. **권한-책임 일치**: 의사결정 권한과 결과 책임 일치
4. **커뮤니케이션 효율**: 보고 단계 최소화 (Flat Organization)
5. **유연성**: 환경 변화에 대응 가능한 구조

### 정원산정 방법론
- 업무량 기반 (Workload Analysis)
- 벤치마킹 기반 (유사 기업 비교)
- 비율 기반 (매출/인원 비율)
- 관리 계층 기반 (Span of Control)

## 조직문화 진단

### 진단 도구
| 도구 | 특징 | 적용 |
|------|------|------|
| OCAI (Cameron & Quinn) | 경쟁가치 모델, 4가지 문화유형 | 문화 유형 파악 |
| Denison 모델 | 적응성, 미션, 일관성, 참여 | 문화 성숙도 |
| 자체 설문 | 프로젝트 맞춤형 | 특정 이슈 진단 |

### 문화유형 (OCAI)
- **관계문화 (Clan)**: 가족적, 협력, 참여
- **혁신문화 (Adhocracy)**: 창의, 도전, 유연
- **시장문화 (Market)**: 성과, 경쟁, 목표
- **위계문화 (Hierarchy)**: 안정, 절차, 통제

## 변화관리

### Kotter 8단계 모델
1. 위기감 조성 (Urgency)
2. 변화 주도 연합 구성 (Coalition)
3. 비전과 전략 수립 (Vision)
4. 비전 전파 (Communication)
5. 장애물 제거 (Empowerment)
6. 단기 성과 창출 (Quick Wins)
7. 성과 확대 (Consolidation)
8. 문화 정착 (Anchoring)

### ADKAR 모델
- Awareness (인식): 왜 변화가 필요한가
- Desire (욕구): 변화에 참여하고 싶은가
- Knowledge (지식): 어떻게 변화할 것인가
- Ability (능력): 변화를 실행할 수 있는가
- Reinforcement (강화): 변화를 유지할 것인가
