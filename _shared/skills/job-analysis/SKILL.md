# 직무분석 스킬

## 용도
NCS(국가직무능력표준) 기반 직무분석, 직무기술서 작성, 역량체계 도출, 직무평가 등 **직무분석 전반의 컨설팅 작업**에 사용합니다. 한국 공공기관 및 민간기업의 직무분석 실무 표준을 따릅니다.

## 적용 조건
- "직무분석 해줘", "직무기술서 작성", "역량모델링" 등의 요청
- NCS 기반 직무 분류체계 구축 시
- 직무평가, 보상체계 설계의 기초 데이터 도출 시

---

## 직무분석 프레임워크 (5단계)

### 1단계: 사전 준비
**목표**: 분석 범위 확정, 기초자료 수집

| 활동 | 상세 | 산출물 |
|------|------|--------|
| 범위 확정 | 분석 대상 조직/부서/직종 결정 | 분석 범위 정의서 |
| 기초자료 수집 | 조직도, 정원표, 기존 직무기술서, 업무매뉴얼 | 수집 자료 목록 |
| NCS 사전 매핑 | 해당 직종의 NCS 대분류→세분류 예비 매핑 | NCS 매핑 초안 |
| 분석 계획 수립 | 일정, 인터뷰 대상, 설문 설계 | 분석 실행 계획서 |

### 2단계: 직무 분류체계 구축
**목표**: 직무군 → 직무 → 직무요소(Task) 위계 수립

**분류 단계**:
```
직무군(Job Family) → 직무(Job) → 직무요소(Task) → 작업활동(Activity)
```

**NCS 분류체계 참조**:
```
대분류(24) → 중분류(80) → 소분류(257) → 세분류(1,022) → 능력단위(~10,000)
```

**매핑 원칙**:
- 1개 직무는 1개 이상의 NCS 세분류에 대응
- 능력단위 수준에서 실제 수행 업무와 매칭
- 매칭률이 50% 미만이면 자체 능력단위 신규 개발 검토
- NCS DB(ncs.go.kr) 최신 버전 기준 매핑

### 3단계: 직무정보 수집
**목표**: 현장의 실제 직무 수행 정보 수집

#### 인터뷰 방법
| 유형 | 대상 | 인원 | 시간 | 목적 |
|------|------|------|------|------|
| 1:1 심층 인터뷰 | 현직자(우수수행자) | 직무당 2-3명 | 60-90분 | Task 도출, 상세 수행 내용 |
| 1:1 심층 인터뷰 | 관리자/팀장 | 직무당 1-2명 | 60분 | 역할 범위, 성과 기준 |
| FGI(Focus Group) | 현직자 그룹 | 4-6명 | 90-120분 | Task 검증, 중요도 합의 |
| SME 자문 | 외부 전문가 | 2-3명 | 60분 | 업계 표준, 미래 직무 변화 |

#### 인터뷰 가이드 템플릿
```
[인터뷰 가이드]
1. 귀하의 주요 업무를 크게 5-7가지로 구분해 주세요.
2. 각 업무(Task)에 대해:
   - 무엇을 하시나요? (What)
   - 어떻게 하시나요? (How — 절차, 도구, 시스템)
   - 얼마나 자주 하시나요? (빈도: 매일/주간/월간/분기/연간)
   - 이 업무가 얼마나 중요한가요? (1-5점)
   - 이 업무의 난이도는? (1-5점)
3. 이 직무를 잘 수행하려면 어떤 지식/기술/태도가 필요한가요?
4. 최근 2-3년 사이 업무에서 크게 달라진 점이 있나요?
5. 향후 이 직무에서 새로 필요해질 역량은 무엇인가요?
```

#### 설문조사 설계
- **조사 대상**: 해당 직무 수행자 전수 또는 대표 표본
- **핵심 문항**: 각 Task별 빈도(Frequency), 중요도(Importance), 난이도(Difficulty) — FID 3축 평가
- **척도**: 리커트 5점 척도 (1=매우 낮음 ~ 5=매우 높음)
- **추가 문항**: 소요 시간 비율(%), 필요 역량, 교육 필요도
- **수집 도구**: 구글폼, 네이버폼, 또는 자체 설문 시스템

### 4단계: 직무기술서 작성
**목표**: 수집 데이터 기반 공식 직무기술서 작성

#### 직무기술서 항목 구조
| 항목 | 내용 | 예시 |
|------|------|------|
| 직무명 | 공식 직무 명칭 | 인사기획 |
| 직무 목적 | 1-2문장 직무 존재 이유 | 조직의 인적자원 확보·개발·유지를 위한 인사제도 기획 및 운영 |
| 소속 | 부서/팀 | 경영지원본부 인사팀 |
| 직급/등급 | 해당 직급 범위 | 대리~과장 |
| 직무요소(Task) | 5-10개 핵심 업무 | 인사제도 기획, 채용관리, 평가운영, 보상설계 등 |
| 작업활동(Activity) | Task별 3-5개 세부활동 | 채용계획 수립, 채용공고 작성, 면접 운영 등 |
| KSA | 필요 지식/기술/태도 | 노동법, HRIS 활용, 커뮤니케이션 역량 |
| NCS 연계 | 매핑된 NCS 능력단위 | 02030201_인사기획 |
| 수행 기준 | 성과 기대수준 | 채용 소요기간 30일 이내, 이직률 10% 이하 |

#### FID 분석 (빈도-중요도-난이도)
- **우선순위 산출**: Task 우선순위 = 빈도(F) × 중요도(I) 가중평균
- **교육 니즈**: 중요도 높 + 난이도 높 = 최우선 교육 대상
- **시각화**: FID 버블차트 (X=빈도, Y=중요도, 크기=난이도)

### 5단계: 역량체계 도출
**목표**: 직무별 필요 역량 체계화

#### 역량 구분
| 역량 유형 | 정의 | 예시 |
|----------|------|------|
| 공통역량 | 전 직원 필수 | 조직이해, 커뮤니케이션, 윤리의식 |
| 리더십역량 | 관리자 필수 | 전략적 사고, 팀 관리, 의사결정 |
| 직무역량 | 특정 직무 필수 | 노동법 지식, 데이터 분석, 설계 역량 |

#### 역량 수준 정의 (4단계)
| 수준 | 정의 | 행동지표 예시 |
|------|------|-------------|
| L1 (기초) | 감독 하에 기본 업무 수행 | 매뉴얼에 따라 업무 처리 가능 |
| L2 (적용) | 독립적으로 업무 수행 | 일상적 업무를 자율적으로 처리 |
| L3 (심화) | 복잡한 상황에서 판단·해결 | 비정형 이슈에 대한 대안 제시 |
| L4 (전문) | 타인을 지도하고 혁신 주도 | 제도 설계, 후배 육성, 개선 주도 |

## NCS 활용 상세

### NCS 검색 및 활용
- **검색**: ncs.go.kr → NCS 및 학습모듈 검색
- **활용 범위**: 능력단위 코드, 능력단위 요소, 수행준거, 지식/기술/태도
- **버전 관리**: NCS는 매년 개정되므로 최신 버전 확인 필수
- **한계 인식**: NCS가 모든 직무를 커버하지 않음 → 자체 보완 필요

### NCS 기반 훈련과정 연계
직무분석 결과를 훈련과정 개발에 연계:
1. 직무별 필수 능력단위 선정
2. 능력단위별 학습모듈 매핑
3. 교육과정 로드맵 설계 (직급별, 경력별)
4. 평가기준 설계 (능력단위 수행준거 기반)

## 주의사항
- 인터뷰 전 **사전동의** 및 비밀보장 안내 필수
- 직무분석 결과가 인사평가·보상에 직결되면 **노조/근로자 대표 의견 수렴** 필요
- 설문 응답률 60% 미만이면 대표성 문제 → 추가 독려 또는 보완 조사
- 직무기술서는 **현재 수행 업무** 기준 (미래 희망 업무가 아님)
- 산출물 작성 시 pptx/docx/xlsx 스킬 연계 사용
